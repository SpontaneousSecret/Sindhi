## 2. PROJECT SCOPE

The Service Provider agrees to design and develop a web-based e-commerce platform with the following core features as agreed upon:

### 2.1. Product Catalog
-   Listing of approximately **400 Store Products**.
-   Dynamic filtering and categorization (Sweet, Namkeen, Dry Fruits).
-   Search and "Load More" pagination functionality.

### 2.2. User Features
-   Responsive Design (Mobile/Desktop).
-   Shopping Cart functionality.
-   Checkout process includes **Cash on Delivery (COD)** and **Online Payments** via Axis Bank Payment Gateway integration.

### 2.3. Admin Dashboard
-   Secure Login for Super Admin.
-   Inventory Management (Add/Edit/Delete Products).
-   Order Management View (Order Status tracking).

## 3. DELIVERY & LOGISTICS MODEL

The website will include specific logic to handle the Client's delivery operations:

### 3.1. Standard Delivery Options
The system will support the following time-slot based delivery options:
-   **3 Hours** Delivery
-   **6 Hours** Delivery
-   **9 Hours** Delivery

### 3.2. Relaxed Delivery (Next Day)
-   Orders placed before 3:00 PM: Eligible for same-day delivery logic (subject to slot availability).
-   Orders placed after 3:00 PM: Delivered the following day (Turnaround Time: Within 1 Day).

### 3.3. Express Delivery
-   **Timeline**: Delivery within 1 Hour.
-   **Surcharge**: An additional fee applies based on the delivery location (geolocation/distance calculation or zone-based logic as defined in technical specs).
